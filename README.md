
# Rich Text Editor

A simple Rich Text Editor built with React, allowing users to format text with bold, italic, underline, strikethrough, and lists.


## Features

- Light/dark mode toggle
- Bold, Italic, Underline, Strikethrough Formatting
- Bullet Lists
- AI Text Suggestions
- Live Preview
